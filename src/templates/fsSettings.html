To determine these values are, navigate to you spaces bucket in DigitalOcean, and copy the "Origin Endpoint", located at the top right.
The URL will look something like this:

https://[bucket-name].[region].digitaloceanspaces.com

The API Endpoint is this URL, without the bucket name, like this:

https://[region].digitaloceanspaces.com

The "Region" and "Bucket" settings below are the corresponding [region] and [bucket-name] values.

To be able to set the "Base URL" above, you need to enable the "Files in this filesystem have public URLs" lightswitch.
The value should be the whole "Origin Endpoint" URL. Do not add the subfolder from the "Subfolder" setting below, if you've added one.

If you want to use the DigitalOcean Spaces CDN, you need to enable it in the Settings for your bucket in DigitalOcean. You will then be able to see the "CDN Endpoint"-URL by selecting it in the dropdown in the top right, where you found the origin URL. This value should then be used as the Base URL above, while all the other settings below should be as described above.
